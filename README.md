# Get pic for 4chan.org

this can download pics from 4chan.org , 1~10 page.

get

```
git clone https://github.com/pmthink/get_pic_for_4chan.git
```
install

```
cd get_pic_for_4chan
pip3 install -r install.txt
```
run

```
python3 get.py
```

if you use linux

you can :

```
python3 get2.py
```

it so fast
